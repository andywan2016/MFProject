the python project is used for data loading, a yahoo finance module is required, you can install as followed:

$ git clone git://github.com/lukaszbanasiak/yahoo-finance.git
$ cd yahoo-finance
$ python setup.py install



For using this part:
Simply use follow command: python YahooFinance.py -f 30, where 30 is how many days look back. The list of stock ticket is at line13, you can add stock you want to the list. line 14 is the output path, you can also change it to your path of file output
